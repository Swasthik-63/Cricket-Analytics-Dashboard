# IPL Cricket Analytics Dashboard

## Project Overview

This project analyzes IPL (Indian Premier League) cricket data from 2008–2024 using Python, Pandas, Jupyter Notebook, and Power BI.

The goal is to extract meaningful insights about team performance, player achievements, and venue statistics, and present them through an interactive dashboard.

## Tools Used

* Python
* Pandas
* Jupyter Notebook
* Power BI

## Key Insights

* Top Winning Teams in IPL
* Top IPL Run Scorers
* Player of the Match Analysis
* Most Active IPL Venues
* Season Summary Metrics
